# Options

- `-c,--config`

 Specify a config file. See [configFiles](</BFC/configFiles>).

- `-e --execute`

 Execute program as well.

- `-i --info`

 Print info about BFC.

- `-o`

 Specify the filename of the resulting executable.

 Args:

 filename

- `-r --remove-executable`

 Delete executable after compiling (only useful with `-e`)

- `-v,--verbose`
 
 Verbose output.

- `--version`

 Print version.
